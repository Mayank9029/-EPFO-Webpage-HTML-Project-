# -EPFO-Webpage-HTML-Project-
This project is a static web page inspired by the official Employees’ Provident Fund Organisation (EPFO)website. 
It was built entirely using HTML and CSS to replicate the layout, design, and content structure of the EPFO homepage — including marquees, menus, social media sections, and an informational footer.
# Overview
The EPFO Webpage project aims to:

Demonstrate front-end development skills using pure HTML and CSS

Create a visually similar layout to a government website

Display official-like information sections such as:

Helpdesk info

Scrolling notifications

Service menus and news updates

Social media links

About EPFO section

Footer navigation and organizational details
